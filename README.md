# Prophecy Tracker

A leaderboard and reward tracker for Prophecy Social. It features real-time data fetching, detailed trader insights

## Tech Stack & Frameworks

- **Core Framework:** React 19 + TypeScript + Vite 8
- **Styling:** Tailwind CSS v4
- **Animations & Scrolling:** Framer Motion + Lenis (for smooth scrolling)
- **Data Visualization:** Recharts
- **API Client:** Axios

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run in Development Mode:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```
